# meetup-tool 要件定義書

## 概要

名刺交換会での会話を録音し、帰宅後に会話要約と人物名を一覧で確認できるツール。

---

## コア機能

### 1. タイムスタンプ記録（イベント中）

- スマホブラウザから Streamlit アプリにアクセスして使用
- 「タイマー開始」ボタンで計測スタート（スマホの録音アプリと同時に開始）
- 会話が終わるたびに「次の人へ」ボタン 1 タップでタイムスタンプを記録
- タイムスタンプは経過秒数の配列として session_state に保持（シンプル実装）
- 「録音終了」ボタンで最終タイムスタンプを記録して完了

### 2. 文字起こし（帰宅後）

- faster-whisper（ローカル・無料）で音声ファイルを文字起こし
- `language="ja"` 指定必須
- タイムスタンプをもとに会話セグメントを人物ごとに分割

### 3. 名前抽出・要約（帰宅後）

- Claude Haiku API が文字起こしテキストを解析
- 「〇〇と申します」「〇〇さん」等の自己紹介フレーズから名前を抽出
- 会話内容を 2〜3 文で要約
- 精度目安: 名前が明確に発話された場合 80〜90%

### 4. 結果表示（帰宅後）

- 処理結果を表形式で一覧表示
- 各行: 番号 / 名前 / 会話要約
- CSV ダウンロード機能あり

---

## オプション機能（v2）

- 名刺写真を撮影して人物データに紐付け
- Claude Haiku Vision で名前・メールアドレスを OCR（精度ほぼ 100%）
- 名刺なしの人はスキップ可能

---

## 技術スタック

| レイヤー       | 技術                    |
| -------------- | ----------------------- |
| UI             | Streamlit（スマホ対応） |
| 音声処理       | faster-whisper          |
| AI 処理        | Claude Haiku API        |
| 名刺 OCR（v2） | Claude Haiku Vision     |

### モデル設定

- Whisper: `small`（精度とスピードのバランス、日本語対応）
- Claude: `claude-haiku-4-5-20251001`

---

## 非機能要件

- スマホブラウザで動作すること（同一 Wi-Fi 経由でローカル PC に接続）
- カット操作は片手・1 タップで完結
- 処理は帰宅後（リアルタイム不要）
- データはセッション内のみ保持（ブラウザを閉じると消去）

## チケット管理

機能ごとのチケットは `Docs/` 配下に連番マークダウンで管理する。

```
meetup-tool/
├── CLAUDE.md          # 本要件定義書
├── app.py             # Streamlit アプリ（全機能 1 ファイル）
├── requirements.txt   # 依存パッケージ
└── Docs/
    ├── ticket-001-timestamp.md        # タイムスタンプ記録
    ├── ticket-002-transcription.md    # 文字起こし
    ├── ticket-003-name-extraction.md  # 名前抽出・要約
    ├── ticket-004-result-display.md   # 結果表示・CSV
    └── ticket-005-business-card-ocr.md# 名刺 OCR（v2）
```

### TODO 管理ルール

各チケットの TODO リストは以下のルールで運用する。

| 状態       | 記法                              |
| ---------- | --------------------------------- |
| 未完了     | `- [ ] タスク内容`                |
| 完了       | `- [x] タスク内容（X）`          |

**完了したタスクは行末に `（X）` を付けること。**

```markdown
# 例
- [ ] 未着手のタスク
- [x] 完了したタスク（X）
```

## 起動方法

```bash
cd /Users/development/claude/meetup-tool
pip install -r requirements.txt
streamlit run app.py
```

スマホからのアクセス: `http://[PC の IP アドレス]:8501`
