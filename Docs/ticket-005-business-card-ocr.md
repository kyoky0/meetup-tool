## 仕様詳細

### 使用モデル

`claude-haiku-4-5-20251001`（Vision 対応）

### プロンプト構造（案）

```
この名刺から以下をJSONで抽出してください:
- name: 氏名
- company: 会社名
- title: 役職
- email: メールアドレス
- phone: 電話番号
不明な項目は null にしてください。
```

### 対応画像フォーマット

- JPG / PNG / WebP / HEIC（スマホ撮影を考慮）

### マージ優先順位

| 項目         | 優先ソース |
| ------------ | ---------- |
| 名前         | OCR（名刺が高精度）|
| 会話要約     | Whisper（名刺に記載なし）|
| メール・電話 | OCR のみ   |

### UI フロー（案）

```
結果テーブル
└── 各行に「名刺を追加」ボタン
    └── 画像アップロード → OCR → テーブル更新
```
