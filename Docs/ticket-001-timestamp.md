# Ticket-001: タイムスタンプ記録

## 概要

名刺交換会のイベント中、スマホブラウザから会話の区切りをタップ 1 回で記録する機能。
録音はスマホの録音アプリ側で行い、本機能はタイミングの記録のみを担う。

## 仕様詳細

### タイムスタンプの構造

```
timestamps = [0.0, cut1, cut2, ..., end]
```

- `timestamps[0]` は常に `0.0`（録音開始）
- 「次の人へ」押下ごとに経過秒数を追記
- 「録音終了」押下で末尾に追記
- 人物数 = `len(timestamps) - 1`

### UI 状態遷移

```
[初期状態] → タイマー開始 → [録音中] → 次の人 × N → 録音終了 → [完了]
```

## 備考

- タイムスタンプはセッション内のみ保持（ブラウザを閉じると消去）
- スマホの録音アプリとタイマーを同時に開始する運用を前提とする
