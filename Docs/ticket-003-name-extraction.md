# Ticket-003: 名前抽出・要約

## 概要

人物ごとの文字起こしテキストを Claude Haiku API に送信し、
相手の名前と会話要約を自動生成する機能。

## 仕様詳細

### 使用モデル

`claude-haiku-4-5-20251001`

### プロンプト構造

```
名刺交換会での会話の文字起こしです。
<会話>
{transcript}
</会話>

以下をJSONのみで返してください:
- name: 相手の名前（「〇〇と申します」等から抽出。不明なら「不明」）
- summary: 会話内容の要約（2〜3文）
{"name": "...", "summary": "..."}
```

### 名前抽出の対象フレーズ（例）

- 「〇〇と申します」
- 「〇〇と言います」
- 「〇〇さん」（他者から呼びかけられた場合）
- 「私は〇〇です」

### 精度目安

| 条件                     | 精度      |
| ------------------------ | --------- |
| 名前が明確に発話された   | 80〜90%   |
| 名前が不明確 / 未発話    | 「不明」  |
