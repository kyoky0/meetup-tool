## デザイン仕様

### カラー

| 用途             | 値                  |
| ---------------- | ------------------- |
| ヘッダー背景     | グラデーション（`#667eea → #764ba2`）|
| バッジ背景       | `#e8f4fd`           |
| バッジ枠線       | `#4a90d9`           |
| バッジ文字       | `#2c5282`           |
| タイマー文字     | `#1a202c`           |

### ボタン

| 種別         | 高さ  | フォント   | 備考            |
| ------------ | ----- | ---------- | --------------- |
| 通常         | 72px  | 20px       |                 |
| プライマリ   | 72px  | 22px・太字 | 次の人へ・処理開始 |

### タイマー

- フォントサイズ: 48px
- フォントファミリー: monospace
- 毎秒自動更新（`time.sleep(1)` + `st.rerun()`、録音中のみ）

---

## 備考

- 自動更新は `recording=True` の時のみ実行（Tab2 の操作に影響しないよう注意）
- ファイルアップローダーの state は Streamlit が内部で保持するため rerun の影響なし
